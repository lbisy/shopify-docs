🔹 小结

完全免费 + 嵌入网页的：

Tawk.to（简单留言工单）

Crisp / JivoChat（轻量聊天 + 工单）

开源系统（Zammad、osTicket、UVdesk）

专业工单 + 网页嵌入：

Freshdesk（免费版可用，坐席少）

Zendesk（试用 14 天，收费）

选择建议：

不需要实时聊天，只要用户提交问题 → Freshdesk Widget 或开源系统最佳。

需要聊天 + 离线工单 → Tawk.to / Crisp / JivoChat 最方便。

技术团队可以维护 → 自建 Zammad / osTicket / UVdesk，长期成本最低。

# Tabs & Accordions（选项卡与手风琴）

**功能概览**  
分组显示内容，可折叠或切换，提升页面可读性。

**功能特点**  
- 支持选项卡和手风琴布局  
- 可自定义标题、内容  
- 支持图标、动画  
- 响应式设计  

**使用步骤**  
1. 拖入 **Tabs & Accordions** 小部件  
2. 添加选项卡或手风琴项  
3. 填写标题和内容  
4. 设置布局、图标和动画  
5. 点击 **更新**  

**参数说明**

| 参数 | 说明 | 可选值 | 默认值 |
|------|------|--------|--------|
| 布局 | Tabs / Accordion | tabs / accordion | tabs |
| 默认展开项 | 默认显示第几项 | 数字 | 1 |
| 图标 | 每项可设置图标 | - | 无 |
| 动画效果 | 内容切换动画 | none / fade / slide | none |

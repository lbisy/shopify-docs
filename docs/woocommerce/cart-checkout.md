# WooCommerce 购物车与结账页

**功能概览**  
自定义购物车与结账页面，优化用户体验。

**功能特点**  
- 可设置购物车布局  
- 支持结账字段自定义  
- 支持优惠码和多种支付方式  

**使用步骤**  
1. 后台 > WooCommerce > 设置 > 购物车 / 结账  
2. 配置页面布局和结账字段  
3. 保存更改  

**参数说明**

| 参数 | 说明 | 可选值 | 默认值 |
|------|------|--------|--------|
| 布局 | 购物车页布局 | standard / compact | standard |
| 支付方式 | 支持的支付网关 | PayPal / Stripe / COD | PayPal |
| 结账字段 | 自定义显示字段 | - | 默认 |
| 优惠码 | 启用 / 禁用 | yes / no | yes |
